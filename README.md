Container Based Automation Solution

Overview:
This project demonstrates a complete cloud-native microservices deployment using AWS services, Docker, Kubernetes, Jenkins, and Infrastructure as Code.

Technologies Used:
Python (Flask)
Docker
Amazon ECR
Amazon EKS
Jenkins
DynamoDB
CloudFormation
CloudWatch
Amazon S3
Amazon Athena

Services:
Product Service – manages product data
Order Service – manages order data
CI/CD Pipeline
Source control: GitHub
Build & deployment: Jenkins
Container registry: Amazon ECR
Orchestration: Amazon EKS

Infrastructure:
Provisioned using AWS CloudFormation
Includes VPC and DynamoDB tables
Monitoring & Analytics
CloudWatch dashboard for DynamoDB metrics
Athena queries on S3-stored JSON data

Key Learnings:
Microservices architecture
Kubernetes deployment
Secure CI/CD pipelines
AWS service integration
Infrastructure as Code
